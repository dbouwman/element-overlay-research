    Scrolling!

    If you scroll when this modal is open, you will see the content behind this scroll up.

    The esri-global-nav example actually modifies the head with a data-header-is-open=true which then applies position:fixed as well as setting the height and width to the current viewport.

    This approach is ok if we're just hacking stuff together, but to implement this pattern in Ember, or some other framework is not friendly.

    GlobalNav watches for window resize events, and sets css properties which are then used in the height/width.

    When we apply position:fixed to almost anything using the bootstrap layout system, it all breaks.
